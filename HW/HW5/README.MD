Прогнозирование стоимости жилья: сравнение пяти регрессионных моделей
1. Обзор проекта
Проект посвящён построению и сравнению моделей машинного обучения для предсказания рыночной стоимости жилых домов. Используется датасет House Sales Prediction с Kaggle, содержащий информацию о продажах домов в округе Кинг (США, штат Вашингтон). Цель – разработать регрессионную модель, способную с высокой точностью оценивать цену дома на основе его характеристик.

В работе реализованы и протестированы пять алгоритмов регрессии:

CatBoostRegressor

XGBoostRegressor

GradientBoostingRegressor

RandomForestRegressor

HuberRegressor

Для каждой модели выполнен подбор гиперпараметров с помощью GridSearchCV (5‑кратная кросс‑валидация). Качество оценивается на тестовой выборке по метрикам MSE, MAE и R² (все метрики рассчитываются для логарифмированной цены, что снижает влияние выбросов).

2. Данные
2.1 Исходный датасет
Датасет kc_house_data.csv содержит 21 613 записей и 21 колонку. Ключевые поля:

Поле	Описание

sqft_above	Площадь дома над землёй
sqft_basement	Площадь подвала

Пропуски в данных отсутствуют.

2.2 Предобработка
Целевая переменная: распределение цен сильно скошено, поэтому применено логарифмирование (np.log1p). Все дальнейшие метрики рассчитываются для логарифмированной цены.

Дата: столбец date преобразован в формат datetime, из него выделены год и месяц продажи, исходный столбец удалён.

Разделение данных: выборка случайным образом разделена на обучающую (80%) и тестовую (20%) с фиксированным random_state=42 для воспроизводимости.

Масштабирование: все признаки стандартизированы (приведены к нулевому среднему и единичному стандартному отклонению) с помощью StandardScaler. Это важно для корректной работы линейных моделей (включая HuberRegressor) и может улучшить сходимость градиентного бустинга.

4. Результаты
Все модели оценивались на тестовой выборке. Для удобства метрики приведены для логарифмированной цены.

Модель	MSE (лог-цена)	MAE (лог-цена)	R² (лог-цена)
CatBoost	0.02	0.11	0.9127
XGBoost	0.03	0.11	0.9099
GradientBoosting	0.03	0.12	0.9078
HuberRegressor	0.03	0.12	0.8911
RandomForest	0.03	0.13	0.8906
Наилучший результат показал CatBoost, достигнув R² ≈ 0.913. XGBoost и GradientBoosting показали очень близкие значения, особенно по MAE. HuberRegaster (устойчивый к выбросам) немного уступил, но всё же превзошёл случайный лес.

Для сравнения также была обучена обычная линейная регрессия (с масштабированием). Её результаты оказались хуже: MSE ≈ 0.04, R² ≈ 0.88. Это подтверждает, что данные имеют нелинейные зависимости, которые ансамблевые методы улавливают лучше.

5. Важность признаков
Анализ важности признаков (feature importance для древовидных моделей и абсолютные значения коэффициентов для HuberRegressor) показал, что наибольшее влияние на цену оказывают:

sqft_living – жилая площадь

grade – оценка качества конструкции и дизайна

lat – широта (географическое положение)

sqft_living15 – средняя жилая площадь соседей

view – качество вида

Эти результаты интуитивно понятны и согласуются с экономикой недвижимости.

6. Как запустить проект
Клонируйте репозиторий (или скачайте файл ноутбука).

Установите необходимые библиотеки:

bash
pip install pandas numpy matplotlib seaborn scikit-learn catboost xgboost kaggle
Настройте Kaggle API:

Загрузите свой файл kaggle.json (инструкция на Kaggle).

В первой ячейке ноутбука предусмотрена загрузка этого файла; следуйте инструкциям (если файл не найден, будет предложено загрузить его вручную).

Запустите ноутбук последовательно. Он автоматически скачает датасет, выполнит всю предобработку, обучит все пять моделей с подбором гиперпараметров и выведет результаты.

Примечание: обучение с полным перебором параметров может занять от 5 до 15 минут в зависимости от аппаратного обеспечения. Для HuberRegressor подбор выполняется быстрее всего.

7. Выводы
Задача прогнозирования стоимости жилья успешно решена с использованием пяти различных регрессионных алгоритмов.

Модели на основе градиентного бустинга (CatBoost, XGBoost, GradientBoosting) показали наилучшие результаты, причём CatBoost оказался лидером.

Устойчивый к выбросам HuberRegressor продемонстрировал результаты, близкие к бустингу, и значительно превзошёл обычную линейную регрессию, подтверждая наличие нелинейностей и выбросов в данных.

Случайный лес, хотя и уступил бустингу, остаётся хорошей базовой моделью.

Ключевыми факторами, определяющими цену дома, являются его размер, качество и местоположение.

Полученные модели могут быть использованы для оценки рыночной стоимости недвижимости или как основа для дальнейших исследований.
