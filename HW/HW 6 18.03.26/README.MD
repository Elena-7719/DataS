# Анализ оттока клиентов телекоммуникационной компании (Telco Customer Churn)

## Описание проекта
В данном проекте проводится исследование данных клиентов телекоммуникационной компании с целью выявления групп клиентов со схожими характеристиками (кластеризация) и обнаружения аномалий. Для этого используются методы машинного обучения: предобработка данных, Isolation Forest для поиска аномалий, KMeans для кластеризации и различные метрики для выбора оптимального числа кластеров.

## Данные
Используется публичный датасет [Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn), содержащий информацию о 7043 клиентах, включая демографические данные, услуги, которыми они пользуются, и информацию об оттоке (Churn).

## Основные этапы работы

### 1. Загрузка и первичный анализ данных
- Загрузка данных через Kaggle API.
- Удаление столбца `customerID`.
- Преобразование столбца `TotalCharges` в числовой тип и заполнение пропусков медианой.
- Разделение признаков на бинарные и категориальные с несколькими значениями.
- Кодирование бинарных признаков с помощью `LabelEncoder`.
- One-hot кодирование категориальных признаков с несколькими категориями.
- Масштабирование всех признаков с помощью `StandardScaler`.

### 2. Поиск аномалий с Isolation Forest
- Для выявления выбросов используется алгоритм Isolation Forest с параметром contamination=0.05.
- Результаты визуализируются на графике зависимости `MonthlyCharges` от `tenure`.

### 3. Кластеризация методом KMeans
- Для подбора оптимального числа кластеров рассматриваются значения k от 2 до 10.
- Для каждого k вычисляются:
  - Инерция (метод локтя)
  - Коэффициент силуэта
  - Индекс Дэвиса-Болдина
  - Индекс Калински-Харабаса
- Оптимальное число кластеров выбрано на основе максимального значения коэффициента силуэта.
- Финальная кластеризация выполняется с оптимальным k, результаты визуализируются с помощью PCA (снижение размерности до 2 компонент).

## Результаты
- Оптимальное число кластеров для KMeans: **2**.
- Визуализация кластеров показала разделение клиентов на две основные группы.
- Isolation Forest выявил около 5% аномальных наблюдений, которые могут представлять собой необычные паттерны поведения клиентов.
