Diabetes Risk Analysis

Описание проекта

Анализ факторов риска развития диабета на основе синтетических данных пациентов. Проект включает загрузку данных, создание базы данных SQLite, анализ и визуализацию ключевых показателей.

Основные функции

· Загрузка данных с Kaggle API
· Создание базы данных SQLite
· Анализ влияния различных факторов на риск диабета:
  · Возраст
  · Индекс массы тела (BMI)
  · Уровень глюкозы
  · Артериальное давление
  · Наследственность
  · Курение
  · Физическая активность

Визуализации

· Распределение пациентов по потреблению калорий
· Корреляционная матрица факторов риска
· Анализ риска в зависимости от веса

Технологии

· Python
· SQLite
· Pandas
· Matplotlib
· Seaborn
