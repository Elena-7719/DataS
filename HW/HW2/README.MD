
 Проект: Анализ данных (EDA) на примере датасета Tips

Этот проект представляет собой учебный пример по загрузке, предобработке и визуализации данных.  
Основная цель — продемонстрировать модульный подход к анализу данных на примере классического датасета `tips` (чаевые в ресторане).

 Структура проекта

| Файл | Описание |
|------|----------|
| `data_loader.py` | Модуль для загрузки данных из разных источников: CSV, JSON, API |
| `data_processor.py` | Модуль для первичного анализа и обработки пропусков |
| `visualizer.py` | Модуль для визуализации данных (более 20 графиков с `matplotlib` и `seaborn`) |
| `main.py` | Основной скрипт, демонстрирующий работу всех модулей на датасете `tips` |

 Возможности

- Загрузка данных из:
  - локального CSV-файла
  - JSON-файла
  - API (с преобразованием в `pandas.DataFrame`)
- Обработка пропусков:
  - заполнение средним, медианой, модой или строкой `'Unknown'`
- Детальное описание датасета:
  - форма, типы данных, пропуски, статистика
- Визуализация:
  - гистограммы, scatter plot, boxplot, violinplot
  - линейные графики средних значений
  - countplot, корреляционная матрица, FacetGrid

Как запустить

1. Убедитесь, что установлены зависимости:
```bash
pip install pandas numpy matplotlib seaborn requests
```

2. Запустите главный скрипт:
```bash
python main.py
```

Если локальный файл `tips.csv` отсутствует, он будет автоматически загружен из библиотеки `seaborn`.

Что демонстрирует `main.py`

1. Загрузка данных
2. Базовый анализ (info, describe, пропуски)
3. Искусственное создание пропусков и их заполнение разными методами
4. Построение полного набора визуализаций

 Модули

`data_loader.py`
- `load_data_csv(path)`
- `load_data_json(path)`
- `load_data_api(url)`

`data_processor.py`
- `data_discribe(df)`
- `data_fill_mean(df)`, `data_fill_median(df)`, `data_fill_freq(df)`, `data_fill_unkn(df)`

 `visualizer.py`
- Содержит множество функций для построения графиков
- Основная: `plot_all_tips(df)` — запускает все визуализации сразу
